# 🐍 Snake!

A kid-friendly snake game with three themed stages, background music, and sound effects — all in a single HTML file. No build step, no dependencies, no server required.

## Features

- **3 themed stages** — Jungle, Ocean, and Space, each with unique visuals and increasing speed
- **10 foods per stage** to advance, with stage-specific food shapes (apple, fish, star)
- **3 lives** with a brief flash on death and automatic respawn
- **Scoring** — +10 / +20 / +30 points per food eaten by stage
- **Background music** — synthesized per-stage melodies via Web Audio API
- **Sound effects** — stage-specific eat sounds, death wah, stage clear fanfare, victory arpeggio
- **Mute toggle** (🔊/🔇) in the HUD
- **Victory screen** with confetti animation after completing all three stages

## Prerequisites

- A modern web browser with JavaScript enabled (Chrome 66+, Firefox 61+, Safari 14+, Edge 79+)
- A keyboard (arrow keys required — no touch/mobile support)

## Getting Started

1. Download `snake.html`
2. Open it in your browser
3. Press any arrow key to start

**Controls:** Arrow keys only. No installation, no internet connection required.

## Development

The entire game lives in `snake.html` — HTML, CSS, and JavaScript all inline. Open it in a browser and edit with any text editor.
